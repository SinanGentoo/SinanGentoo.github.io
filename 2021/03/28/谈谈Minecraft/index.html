谈谈Minecraft

2021-03-28

　　我是一名 Minecraft 玩家，从 15 年入坑开始算起起码也有 3 年以上的游玩经验了。2019 年可谓是 Minecraft 很不平凡的一年了，ScriptEngine，Minecraft:地下城，Minecraft Earth 似乎都在标志着微软收购后的 Minecraft 时代已经完全不一样了。在 2020 即将来临之际我想谈谈 Minecraft 的一切，包括 Minecraft 的未来展望与我的游玩历史，以此纪念我与 Minecraft 一同度过的时光。

我和 Minecraft

　　要说我怎么入坑的是在 15 年暑假的一次偶然。我在刷腾讯新闻的时候 （当时我还有这癖好?） 看见一篇新闻说有一个人用一款游戏还原了清明上河图。当时我就想哪款游戏还能这样玩？就直接百度了一下 Minecraft，按照了上面的教程下载，打开，然后….就没有然后了🌚在网上找到一个整合包，解压，启动，进入世界。生成完毕后我的第一反应就是 “卡” ！没错，就是卡。我当时用的电脑是 IBM 在卖给联想时发售的最后一款 ThinkPad…先不说笔记本没有独显，处理器过时，整合包里装的大型 mod 直接就让我按下启动的那一刻发出一声惨叫，于是我玩了五分钟就把这游戏丢到一边了…

　　不甘心的我把目光瞄准了正在旁边充电的 iPad，然后就找到了基岩版，之后就真正意义上开始了我的 Minecraft 游玩之旅。（顺带一提，当时因为 Minecraft 要付费我还学会了越狱….）

　　第一次 PE 游玩出生在了一个海滩上。因为我在 Minecraft 之前没有玩过任何 3D 沙盒游戏而且也没看过任何有关 PE 的教程所以我就傻乎乎的点方向键把自己按海里去了…因为当时分不清生存和创造，我以为快要淹死了，但又不知道怎么办。于是我乱点屏幕，在海边把自己用砖块阶梯围了起来，然后我就这样透过缝隙，望着月亮度过了我在 Minecraft 里的第一个夜晚。后来看了一些教程，也学会了基本操作，开始用各种方块建房子，搭过山车玩。

　　后来我有了一台 Android 手机，仍然对 Minecraft 念念不忘，于是接触了多玩我的世界盒子，也发现了各种各样好玩的东西。里面不仅有 Minecraft 教程和实况主直播，极大程度上帮助了我这个小白熟悉了这个游戏；更棒的是里面居然还有地图！我也第一次亲眼看到了别人的世界。而且在这个阶段我开始敢开难度和生存了（之前我看见蜘蛛和僵尸晚上就做噩梦…），也学会了开联机房间和其他人一起玩。后来因为知道了多玩抄袭且开始收费了，我就离开了多玩。就在这个时候发现了一个至今都在帮助我游玩的帖子 —— 由贴吧 ID@defanive2发表的MC1.3 原版生存。这个帖子从 1.3 更新到了 1.7，让我知道了什么叫 Optifine，什么是 Forge，怎么样做刷铁塔等等，我也正式入坑了国际版。之后就开始与其他人一起玩领域，玩 Hypixel，玩多人极限生存，一直到现在。

Minecraft 的未来

　　先谈谈它的热度。Minecraft 从出生开始就获奖不断，也在近期销量超越了俄罗斯方块，成为了世界销量第一的游戏，可见 Minecraft 的魅力。我认为 Minecraft 能做到这种程度和微软的努力离不开关系。当时 Notch 在写 Minecraft 的时候并没有想到会有现在这么流行，所以代码写的也只是给自己看的，注释都不写…等 Minecraft 火了之后，他一个人又忙不过来，肯定需要建立一个工作室去运营，但是没人能看懂他写的代码啊…于是代码就这样越来越慢，后期的更新也越来越慢。当微软收购 Mojang 的时候并没有继续像 Notch 时代一样继续破罐子破摔，而是做了几项工作：大力推进 C++版本的开发，支持 Minecraft 的模组生态与重构 Minecraft:Java Edition 的代码。于是我们都知道的，Minecraft 收购之前跳票不断的 C++版本如同打了鸡血一般疯狂更新，地狱/附魔/红石/酿造系统昨天还有人预言 PE 永远不会有，今天就当场打脸，销量也因此突飞猛进，迅速超过了 Java 版。

　　但 Java 不是那么好重构的，Mojang 基本上是更一个版本优化一点点，Notch 走后 Minecraft 的优化更是降到了谷底。但另外一项计划就发挥了重要的作用：扶持 Mod 生态。当时最通用的 ModLoader：Forge 出了 1.7 版本后 Mod 如同雨后春笋般冒了出来（虽然我很想吐槽 Forge 开发团队的效率，1.8 测试版都出来了 1.7 才刚发 Forge 测试版），很多有名的 Mod 也是在那个时候冒出来的，可以说 Mod 拯救了 Java 版，微软也有更多的时间重构代码，并终于在 1.13 版本大致重构结束了。（Forge 又不出意外的跳票了🌚）

　　我们再来看看大陆。我大中国一向被其他国家认为不在地球上（狗头保命，只是吐槽外国游戏对中国的不友好），Minecraft 的热度相比欧美更是在逐渐下降。微软当然不想看到失去这个世界第二大经济体的收入，于是就联系网易代理了 Minecraft。虽然玩家对网易所谓的“中国特供版”褒贬不一，但从热度上来说微软确实下了一步好棋，所以国内一些玩家所宣传的 Minecraft 药丸是不存在的，只不过圈子大了而已。

写在最后

　　我认为 Minecraft 不仅仅只是一款游戏，正如微信不仅仅是一款聊天软件。Minecraft 更是一种让人释放想象力的途径。你可以在游戏里建造动画里的天空之城，制作童话里的城堡，你还可以还原你的学校，你居住的城市甚至整个世界！Minecraft 更是一种教学途径，利用游戏的方式解释四则运算，解释“采菊东篱下，悠然见南山”，解释重力物质的构成。最后，我祝愿 Minecraft 在 2020 年继续保持蓬勃向上的态势，成为真正意义上全人类的文化遗产。
